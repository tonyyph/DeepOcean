# React Native / Expo Skill

Use strict TypeScript.

Rules:
- No any unless absolutely unavoidable
- Avoid unnecessary null
- Prefer undefined when suitable
- Use react-native-safe-area-context
- Do not use deprecated SafeAreaView from react-native
- Keep components small
- Memoize expensive rendering
- Avoid inline heavy objects in render
- Avoid duplicated logic
- Respect iOS and Android differences
- Avoid adding heavy libraries without justification
