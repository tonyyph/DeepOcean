# Deep Ocean UI UX Skill

Prioritize:
- Clear hierarchy
- Premium spacing
- Consistent cards, buttons, modals, sheets
- Accessible contrast
- Smooth visual depth
- Ocean gradients
- Beautiful empty states
- Large touch targets

Bottom sheets:
- Must stand out from background
- Use dim overlay when needed
- Avoid being pushed too high
- Drag gestures must work
- Test small screens

Never redesign unrelated screens unless requested.
